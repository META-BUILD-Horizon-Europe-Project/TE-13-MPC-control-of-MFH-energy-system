# TE-13-MPC-control-of-MFH-energy-system
iDM’s TE-13 builds on its existing iON energy optimiser originally developed for single-family homes, extending it towards multi-family home operation. EURAC’s TE-13 is a supervisory MPC for MFH buildings that optimises operation of electrified energy systems (Heat pumps, thermal energy storage, DHW, PV and battery).
